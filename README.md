# upventure

### E-commerce application made with MVI/Redux pattern.
### Written entirely in kotlin.

+ [Retrofit for HTTP requests](https://square.github.io/retrofit/)
+ [Hilt for dependency injection](https://dagger.dev/hilt/)
+ [Used Airbnb/epoxy to simplify the process of working with RecyclerViews](https://airbnb.io/projects/epoxy/)
+ [Facebook/Shimmer was used for skeleton loading effect while loading products](https://facebook.github.io/shimmer-android/)
